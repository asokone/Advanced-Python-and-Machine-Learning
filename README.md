# Advanced-Python-and-Machine-Learning
This learning plan provides a hands on introduction to data analysis and machine learning techniques. The Participants will learn Python fundamentals, explore data using NumPy and Pandas, and apply key ML methods such as supervised and unsupervised learning. Through guided modules and practical exercises, they will build skills in model development, evaluation, and automated ML tools like AutoAI. 
 
By the end of this cohort, participants will be able to:
•	Use Python for data analysis, including data types, loops, functions, NumPy, and Pandas.
•	Perform data wrangling, cleaning, and exploratory data analysis (EDA).
•	Understand core machine learning concepts and major learning methods.
•	Build and evaluate supervised and unsupervised ML models.
•	Apply AutoAI tools to automate model development.
